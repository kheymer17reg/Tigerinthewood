## 🎮 Что это за игра?

**Тигрёнок в лесу** - это интерактивная игра для обучения программированию.

Ты пишешь простые команды на русском языке, чтобы помочь тигрёнку найти выход из лесу.

### Примеры команд:
```
вправо()      - движение вправо
вправо(3)     - движение вправо на 3 клетки
вверх()       - движение вверх
есть()        - съесть мясо
взять()       - взять ключ
открыть()     - открыть дверь
```
