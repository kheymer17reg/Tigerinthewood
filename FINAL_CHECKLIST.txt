СПОСОБ 1: Через веб-интерфейс (самый простой)

1. Открой GitHub → https://github.com
2. Создай новый репозиторий "tiger-game"
3. Нажми "Add file" → "Upload files"
4. Перетащи все файлы из папки проекта
5. Напиши сообщение: "Initial commit: Tiger Game"
6. Нажми "Commit changes"
7. Перейди в Settings → Pages
8. Выбери Branch: main
9. Нажми Save
10. Жди несколько минут
11. Получи ссылку: https://твой-ник.github.io/tiger-game/

СПОСОБ 2: Через Git (для продвинутых)

1. Установи Git
2. git clone https://github.com/твой-ник/tiger-game.git
3. Скопируй все файлы в папку tiger-game
4. git add .
5. git commit -m "Initial commit: Tiger Game"
6. git push origin main
