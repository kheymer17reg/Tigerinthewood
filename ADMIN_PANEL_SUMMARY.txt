🐛 РЕШЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════

ПРОБЛЕМА: Кнопка админ-панели не видна
РЕШЕНИЕ:
  1. Открой консоль браузера (F12)
  2. Проверь, нет ли ошибок
  3. Очисти кэш браузера (Ctrl+Shift+Delete)
  4. Перезагрузи страницу (F5)

ПРОБЛЕМА: Пароль не работает
РЕШЕНИЕ:
  1. Убедись, что вводишь admin123
  2. Проверь раскладку клавиатуры (английская)
  3. Нет пробелов в начале/конце

ПРОБЛЕМА: Файл не загружается
РЕШЕНИЕ:
  1. Открой консоль браузера (F12)
  2. Посмотри на ошибки
  3. Проверь формат JSON
  4. Убедись, что есть все обязательные поля

ПРОБЛЕМА: Данные не сохраняются
РЕШЕНИЕ:
  1. Проверь, что localStorage включен
  2. Браузер не в приватном режиме
  3. Очисти кэш браузера
  4. Попробуй в другом браузере
