GitHub кэширует старые файлы!
Нужно полностью очистить и загрузить новую версию!


🚀 СПОСОБ 1: УДАЛИТЬ И СОЗДАТЬ НОВЫЙ (самый простой)
═══════════════════════════════════════════════════════════════

Шаг 1: Удали старый репозиторий
  1. GitHub → Settings → Danger Zone
  2. "Delete this repository"
  3. Введи название для подтверждения
  4. Нажми "Delete"

Шаг 2: Создай новый репозиторий
  1. GitHub → "+" → "New repository"
  2. Назови "tiger-game"
  3. Выбери "Public"
  4. Нажми "Create repository"

Шаг 3: Загрузи НОВЫЕ файлы
  1. "Add file" → "Upload files"
  2. Перетащи файлы:
     - tiger_game_improved.html
     - example_level.json
     - hard_level.json
     - README.md
  3. Сообщение: "Initial commit: Tiger Game v2.1"
  4. "Commit changes"

Шаг 4: Включи GitHub Pages
  1. Settings → Pages
  2. Branch: main
  3. Save
  4. Жди 2-3 минуты

Шаг 5: Очисти кэш браузера
  1. Ctrl+Shift+Delete
  2. "All time"
  3. "Clear data"

Шаг 6: Открой новую ссылку
  https://твой-ник.github.io/tiger-game/tiger_game_improved.html


🚀 СПОСОБ 2: NETLIFY (не кэширует)
═══════════════════════════════════════════════════════════════

Шаг 1: Перейди на Netlify
  https://netlify.com

Шаг 2: Sign up (или войди через GitHub)

Шаг 3: "Add new site" → "Deploy manually"

Шаг 4: Перетащи папку с файлами

Шаг 5: Получи ссылку
  https://твое-имя.netlify.app/tiger_game_improved.html


🚀 СПОСОБ 3: VERCEL (очень быстро)
═══════════════════════════════════════════════════════════════

Шаг 1: Перейди на Vercel
  https://vercel.com

Шаг 2: Sign up (или войди через GitHub)

Шаг 3: "New Project" → "Import Git Repository"

Шаг 4: Выбери репозиторий

Шаг 5: Получи ссылку
  https://твое-имя.vercel.app/tiger_game_improved.html


✅ ПРОВЕРКА ЧТО ЗАГРУЗИЛАСЬ НОВАЯ ВЕРСИЯ
═══════════════════════════════════════════════════════════════

Открой консоль браузера (F12) и проверь:

[ ] Кнопка "🔐 Админ" видна в header
[ ] Вкладка "Песочница" работает
[ ] 4 кнопки в песочнице видны
[ ] Админ-панель работает (пароль: admin123)
[ ] Загрузка файлов работает
[ ] Нет красных ошибок в консоли


🐛 ЕСЛИ ВСЕ ЕЩЕ СТАРАЯ ВЕРСИЯ
═══════════════════════════════════════════════════════════════

1. Полностью очисти кэш браузера
   Ctrl+Shift+Delete → "All time" → "Clear data"

2. Закрой браузер полностью

3. Открой браузер заново

4. Если не помогает, используй приватный режим
   Ctrl+Shift+N (Chrome)
   Ctrl+Shift+P (Firefox)

5. Если все еще не работает, используй Netlify или Vercel
