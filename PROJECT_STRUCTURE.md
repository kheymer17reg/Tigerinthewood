## Как расширить проект?

### Добавить новый уровень
1. Открой `tiger_game_improved.html` в текстовом редакторе
2. Найди объект `levels` в JavaScript
3. Добавь новый уровень:
```javascript
7: {
    name: "Новый уровень",
    start: { x: 0, y: 0 },
    exit: { x: 7, y: 7 },
    objects: [
        { type: 'tree', x: 3, y: 3 },
        ...
    ],
    task: "Описание задания"
}
```

### Добавить новую команду
1. Найди функцию `executeCommand`
2. Добавь новый case:
```javascript
case 'новая_команда':
    return newCommand();
```

### Изменить размер поля
1. Найди `grid-template-columns: repeat(8, 60px)`
2. Измени 8 на нужное число

## Файлы для редактирования

Если хочешь изменить игру:

1. **Добавить уровни** - редактируй объект `levels` в JavaScript
2. **Изменить стили** - редактируй CSS в `<style>`
3. **Добавить команды** - редактируй функцию `executeCommand`
4. **Изменить размер** - редактируй CSS для `.cell` и `#game-grid`
