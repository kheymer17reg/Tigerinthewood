# 🐯 Тигрёнок в лесу - Полная версия

## Что объединено

Эта версия объединяет лучшее из двух версий игры:

### ✅ Из основной версии:
- 🔐 **Авторизация через Telegram бота** (@Tiger42bot)
- 📤 **Система модерации уровней пользователей**
- 🔒 **Смена пароля**
- 🤖 **ИИ помощник**
- 📊 **Расширенная статистика**
- ⚙️ **Админ-панель с редактором уровней**

### ✅ Из подпапки Tigerinthewood-main:
- 🎮 **Улучшенные уровни** (improved_levels.js) - более интересные задания
- 🧪 **test_game.js** - для тестирования

## Новые файлы

- `improved_levels.js` - улучшенные уровни 1-6 с более интересными заданиями
- `test_game.js` - тестовая версия игры
- `auth.js` - система авторизации
- `moderation_system.js` - модерация пользовательских уровней
- `Tgbot/` - Telegram бот для выдачи логинов и паролей

## Как запустить

### 1. Запуск игры
Просто открой `index.html` в браузере

### 2. Запуск Telegram бота (для авторизации)
```bash
cd Tgbot
bash START_BACKGROUND.sh
```

Или на Windows:
```bash
cd Tgbot
START.bat
```

## Что нового в улучшенных уровнях

### Уровень 1: 🌱 Первые шаги в лесу
- Больше деревьев по краям
- Первое мясо как награда
- Связь с историей пробуждения

### Уровень 2: 🍃 Обход препятствий
- Стена деревьев посередине
- Мясо в верхнем проходе
- Учит обходить препятствия

### Уровень 3: 🌿 Лесная тропа
- Диагональный лес
- 3 куска мяса на пути
- Учит использовать числа в командах

### Уровень 4: 🌳 Охота в лесу
- Густой лес по краям
- 4 куска мяса для охоты
- Учит комбинировать движение и еду

### Уровень 5: 🔑 Волшебный ключ
- Лабиринт с дверями
- 2 двери и ключ
- Мясо за дверями

### Уровень 6: 🏆 Большое приключение (ФИНАЛ)
- Большой лабиринт
- 4 куска мяса, 2 ключа, 2 двери
- Использует все выученные команды
- Выход = дом тигрёнка

## Техническая информация

- Python 3.8+ для бота
- Flask для API сервера
- pyTelegramBotAPI для бота
- Vanilla JavaScript для игры
- localStorage для сохранения данных

## Поддержка

Если что-то не работает:
1. Проверь что API сервер запущен
2. Проверь что бот запущен
3. Проверь логи: `tail -f bot.log` и `tail -f api_server.log`
4. Проверь что порт 5000 открыт
