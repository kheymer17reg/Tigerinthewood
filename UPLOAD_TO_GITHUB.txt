🚀 СПОСОБ 1: ЧЕРЕЗ ВЕБ-ИНТЕРФЕЙС (самый простой)
═══════════════════════════════════════════════════════════════

Шаг 1: Открой GitHub
  1. Перейди на https://github.com
  2. Войди в свой аккаунт

Шаг 2: Создай репозиторий
  1. Нажми "+" в правом верхнем углу
  2. Выбери "New repository"
  3. Назови его "tiger-game"
  4. Выбери "Public"
  5. Нажми "Create repository"

Шаг 3: Загрузи файлы
  1. Нажми "Add file" → "Upload files"
  2. Перетащи все файлы из папки проекта
  3. Или нажми "choose your files" и выбери их

Шаг 4: Сохрани изменения
  1. Напиши сообщение: "Initial commit: Tiger Game"
  2. Нажми "Commit changes"

Шаг 5: Включи GitHub Pages
  1. Перейди в Settings
  2. Найди Pages
  3. Выбери Branch: main
  4. Нажми Save
  5. Жди несколько минут

Шаг 6: Получи ссылку
  1. Вернись в Settings → Pages
  2. Найди сообщение "Your site is live at..."
  3. Скопируй ссылку
  4. Добавь имя файла: /tiger_game_improved.html


🚀 СПОСОБ 2: ЧЕРЕЗ GIT (для продвинутых)
═══════════════════════════════════════════════════════════════

Шаг 1: Установи Git
  1. Скачай с https://git-scm.com/download/win
  2. Установи

Шаг 2: Настрой Git
  git config --global user.name "Твое имя"
  git config --global user.email "твой@email.com"

Шаг 3: Клонируй репозиторий
  git clone https://github.com/твой-ник/tiger-game.git
  cd tiger-game

Шаг 4: Скопируй файлы
  1. Скопируй все файлы проекта в папку tiger-game

Шаг 5: Добавь файлы
  git add .

Шаг 6: Сохрани изменения
  git commit -m "Initial commit: Tiger Game"

Шаг 7: Загрузи на GitHub
  git push origin main
